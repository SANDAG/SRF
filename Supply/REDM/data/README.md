hs

hs_sf, hs_mf

hh, hh_sf, hh_mf

emp*indus*, emp*comm_l, emp_office, emp_other*

dev_sf, dev_mf, dev_indus, dev_comm, dev_office

dev, vac, unusable, acres

### dwelling units per acre

DUA

Cap_HS, Cap_HS_SF, Cap_HS_MF
